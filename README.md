# Titanic Survival Prediction

## Dataset
The dataset used in this project is the Titanic dataset, which contains information about passengers such as age, gender, passenger class, and ticket fare.

## Machine Learning Problem
This project addresses a supervised classification problem.

## Target Variable
The target variable is Survived, which indicates whether a passenger survived (1) or not (0).

## Problem Description
The objective of this project is to predict whether a passenger survived the Titanic disaster based on passenger information such as age, gender, gender, passenger class, and ticket fare.
